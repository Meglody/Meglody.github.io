> 经过思考的一版

```javascript
function price(n){
    const priceMap = [
        {min: 1, max: 5, price: 5},
        {min: 6, max: 10, price: 4},
        ...
    ]
    for(const {min, max, price} of priceMap){
        if(n >= min && n <= max){
            return price
        }
    }
}
```
